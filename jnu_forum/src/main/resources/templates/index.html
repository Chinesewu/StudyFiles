Description

河南大学大学生运动安全指导学习论坛，是一个大学生运动安全学习论坛，为喜爱体育安全知识学习者提供一个学习的社区化平台。主要功能有登录、注册、浏览帖子、发帖、上传照片 帖子分类浏览、评论、个人用户的积分、点赞（置为有用评论）、站内信通知、用户积分排行榜等等。它是一个校内讨论交流的平台，师生可以畅所欲言。

Technologies

Java 8 Standard Edition、 Spring MVC、 Spring Data、 Spring Security、 Spring Boot、 MyBatis、 HTML（Thymeleaf）、 CSS（Materialize）、 Maven、 MySql、 Redis

Software tools

Idea、 MySQLWorkbench [MySQL]、PostMan
